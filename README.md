# WOS Wiki — Community Feedback

This is the community review hub for the [WOS Wiki](https://www.digitalrevenuestudio.co.uk/drs-wos-wiki.html) — a free Whiteout Survival guide wiki built by a daily F2P player.

## What is this?

New guides are shared here for community review before going live on the main wiki. The goal is to catch errors, outdated information and missing content before new players rely on it.

## How to review

1. Visit the [review page](https://digitalrevenuestudio.github.io/community-feedback/)
2. Click any guide to read it
3. Use the [Community Hub](https://www.digitalrevenuestudio.co.uk/wos-community.html) feedback form to flag anything wrong
4. Corrections are made fast — usually same day
